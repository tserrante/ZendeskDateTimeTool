Instructions

To insert a date time string in the Zendesk comment field, left click inside the comment box and press ctrl + enter
Right click inside the Zendesk comment field and from the context menu, select "Generate Timeline". Please note this only collects comments that begin with the MM/DD/YY hh:mm: format

There can be a few reasons for this.

  • There are no date time tags in the correct format (MM/DD/YY hh:mm:)
  • The generate timeline function timed out
    • After right clicking into Zendesk's text box there is a backend timer that starts. After five seconds, the timeline process exits and the timeline cannot be created.
    • This does not prevent the timeline function from running again. It only prevents it for that particular right click.
Comments beginning with a tilde character (~) will be omitted from the timeline.
i.e. 01/01/24 14:13: ~comment
